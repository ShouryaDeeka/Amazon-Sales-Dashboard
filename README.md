# 📊 Amazon Sales Dashboard (Excel)

An interactive Amazon Sales Dashboard built in Microsoft Excel to transform raw ecommerce transaction data into meaningful business insights.

This dashboard analyzes over 100,000 orders and provides a comprehensive view of revenue performance, order fulfillment, category contribution, payment behavior, and sales trends through dynamic visualizations and filters.

---

## 🚀 Project Overview

The objective of this project was to create an executive-style dashboard that enables quick decision-making by presenting key business metrics in a clear and interactive format.

The dashboard allows users to:

- Monitor overall business performance
- Analyze revenue trends over time
- Track order fulfillment status
- Evaluate category-wise sales performance
- Understand customer payment preferences
- Filter insights dynamically using slicers

---

## 📈 Key Business Insights

### Revenue Performance
- Generated **$91.8M** in total revenue
- Processed **100K+ orders**

### Category Analysis
- Electronics emerged as the top-performing category
- Generated **$15.58M** in sales revenue

### Sales Trends
- Monthly revenue remained stable between **$1.3M – $1.6M**
- Consistent business growth observed across FY20–FY24

### Order Fulfillment
- **74.6%** of all orders were successfully delivered

### Payment Analysis
- Credit Cards accounted for **35%** of all transactions
- Digital payments contributed to **95%** of purchases
